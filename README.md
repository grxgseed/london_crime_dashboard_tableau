# London Crime Dashboard (Tableau)

## Overview

This project presents an interactive Tableau dashboard analysing borough-level crime data in London. The dashboard explores how crime is distributed across boroughs, categories, and time.

## Objectives

* Visualise crime distribution across London boroughs
* Analyse trends in crime over time
* Identify dominant crime categories
* Enable interactive exploration of the data

## Dashboard Features

* Bar chart showing crime by borough
* Line chart displaying crime trends over time
* Breakdown of crime by category
* Interactive filters for borough and crime type

## Key Insights

* Crime is concentrated in a small number of boroughs
* Certain crime categories account for a large proportion of total incidents
* Crime levels vary over time, suggesting potential seasonal patterns

## Tools Used

* Tableau Public
